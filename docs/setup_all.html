Configuring WSForm

If you do not want this Special page to be shown. Edit config.php and add : 'allow-special-page-setup' => false

Read the manual (readme or Special:WSForm/Docs/wsform_installation) on how to create a user for WSForm and add this to a user group that has the following permission (example is if WSForm user is in group admin: $wgGroupPermissions['admin']['upload'] = true;
$wgGroupPermissions['admin']['uploadfile'] = true;
$wgGroupPermissions['admin']['upload_by_url'] = true;
